In React, we don't actually "change pages" by loading entirely new HTML files from a server. Instead, React uses a **Single Page Application (SPA)** model. It loads one HTML page and uses JavaScript to swap components in and out of view, making the transition nearly instantaneous.

The industry standard way to handle this is using a library called **React Router**.

Here is a quick, modern guide to setting up page navigation in React.

---

## 1. Install React Router

First, you need to add the routing package to your project. Run this in your terminal:

```bash
npm install react-router-dom

```

---

## 2. Set Up Your Routes

You define your "pages" (which are just regular React components) and map them to specific URL paths.

Here is how you set up the router in your `App.jsx` or `main.jsx` file using the modern `createBrowserRouter`:

```jsx
import React from 'react';
import { createBrowserRouter, RouterProvider } from 'react-router-dom';

// 1. Import your page components
import Home from './pages/Home';
import About from './pages/About';
import Contact from './pages/Contact';

// 2. Define the paths and the components that should render
const router = createBrowserRouter([
  {
    path: "/",
    element: <Home />,
  },
  {
    path: "/about",
    element: <About />,
  },
  {
    path: "/contact",
    element: <Contact />,
  },
]);

// 3. Provide the router to your app
function App() {
  return <RouterProvider router={router} />;
}

export default App;

```

---

## 3. How to Navigate Between Pages

### Method A: Using the `<Link>` Component (For User Clicks)

Never use a standard HTML `<a href="...">` tag in React Router. Standard anchors force a full browser reload, which ruins the SPA performance benefit. Instead, use the `<Link>` component.

```jsx
import { Link } from 'react-router-dom';

function Navbar() {
  return (
    <nav>
      {/* 'to' replaces 'href' */}
      <Link to="/">Home</Link> | 
      <Link to="/about">About</Link> | 
      <Link to="/contact">Contact</Link>
    </nav>
  );
}

```

### Method B: Using `useNavigate` (For Programmatic Navigation)

Sometimes you need to change the page automatically after an action happens—like redirecting a user to a dashboard *after* they successfully submit a login form. For this, use the `useNavigate` hook.

```jsx
import { useNavigate } from 'react-router-dom';

function LoginForm() {
  const navigate = useNavigate();

  const handleSubmit = (event) => {
    event.preventDefault();
    
    // ... do your login logic here ...

    // Redirect the user to the dashboard
    navigate('/dashboard');
  };

  return (
    <form onSubmit={handleSubmit}>
      <button type="submit">Log In</button>
    </form>
  );
}

```

---

> 💡 **Quick Tip:** If you want to highlight the "active" link in your navigation bar (e.g., making the "Home" text bold when the user is on the home page), swap out `<Link>` for `<NavLink>`. It automatically adds an `.active` class to the element when the URL matches!
